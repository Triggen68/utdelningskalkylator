# Utdelningskalkylator

Enkel webbapp som räknar månads- och årsutdelning från:

- Aktiekurs (kr)
- Direktavkastning (%)
- Belopp att investera (kr)

## Live

https://triggen68.github.io/utdelningskalkylator/

## Stack

Ren HTML + CSS + JavaScript. Inga byggsteg, inga beroenden.

## Kör lokalt

Öppna `index.html` direkt i en browser, eller starta en lokal server:

```powershell
python -m http.server 8000
```

Öppna sedan http://localhost:8000
